# Security policy

## Repository scope

This is a Markdown-only, non-operational portfolio repository. It contains no
deployable automation, credentials, live service configuration, wallet or pool
configuration, compiled programs, launchers, or third-party media.

## Reporting a concern

Please report a suspected disclosure through GitHub's private vulnerability
reporting feature for this repository. Include the affected file, the exposed
value or behavior, and enough context to reproduce the concern. Do not post
credentials, personal data, or exploit details in a public issue.

## Response priorities

Reports involving exposed secrets, private infrastructure details, financial
automation parameters, or configuration that could enable unauthorized
activity receive the highest priority. Confirmed sensitive material will be
removed from the current revision and repository history as appropriate.

This policy covers the published documentation only. It does not offer support
for recreating, deploying, or operating the systems described here.
